Patch Notes

Below is a list of patch notes for changes made to the website and module. It should be noted some smaller and less relevant changes are not listed. For those curious about every detailed change to the site, the source code and change history is available on GitHub.

Update 1.12.1

  • New Condition Definitions!
    • We've reworked the Conditions in the module to more properly apply to the setting and game balance
  • Several new rules and rule tweaks
    • Advantage/Disadvantage, Doubling/Halving, Flanking, Non-Lethal Strikes, Limited Resources, Weapon Proficiency stacking
    • Also specified Stand summoning/dispelling must be within 2m and Unarmed Strikes get "Open Hand"
  • Lots of new Feats
    • Additional Sense, Aura of Awareness, Close-Range Mobility, Confident Support, Elementalist, Enfeebler, Extra Experience, Focus, Heckler, Overheal, Powerful Presence, Resource Regeneration, Sharp Mind, and Tactician
  • Several new Abilities
    • Custom Bodily Ability, Distant Defense, Doubled Range, Flight
  • A few new weapons
    • Hand Wraps, Meteor Hammers, and Tonfa
  • A number of Feats, Abilities, and weapons were changed slightly as well
    • A complete list of these would be too long to write here, so be sure to check what you have!
  • A few tweaks and additions to weapon attributes
    • The Conductive and Radius attributes were tweaked, and a new Limited Hand attribute was added
  • Some small changes to a few Classes
    • Mostly options for the new Abilities and Feats
    • Ability-Type gets Distant Defense and Resource Regeneration. Also Earth, Wind, and Fire was removed as an example to avoid confusion, replaced by Man in the Mirror.
    • Act-Type gets Distant Defense and Resource Regeneration, and the Long-Range type gets some more starting options
    • Hive-Type gets Resource Regeneration
    • Ranged-Type gets Distant Defense and some more options at Level 1
  • Familiars got some general tweaks
    • Specified their AC bonus applies to their Stands as well
    • Additional Ability Score Increase at level 19
    • More starting Proficiencies and a few tweaked options
    • Some Features and Feats were tweaked slightly, check what you have!
Website Update v1.12.0.1
  • Character Sheet changes
    • The editor was updated to reflect the new character sheet
    • This shouldn't cause any issues with existing characters and data, please let me know if you experience any!
  • Fixed a few links and formatting issues

Update 1.12.0 - The Modular Update

  • The Modular Update!
    • Every Class now has a variant level-up table that allows for more varied options when building your character
    • This system is a Variant Rule, and it's up to the DM if it should be used
  • New Skills
    • The 5e Skills system doesn't quite fit the settings commonly seen in JoJo's, so we've made some changes to most of the Skills to make them fit better.
    • Descriptions of these Skills as well as a guide on how to convert from the 5e versions is in the Rules section
    • The Character Sheet was updated to include these new Skills
  • Act-Type was reworked
    • Even the non-modular version of the Class is more open and versatile now
  • Stand AC was buffed
    • All 3 stats (Precision, Durability, Speed) are now used to calculate it
  • A number of new Feats were added
    • Act Modification, Burst, Class Initiate, Dark Determination, Melee Focus, Object Stand, Piercer, Sentinel, Slasher, Specialized Stand Senses, Stand Breaker, Weapon Attribute Specialist
  • A number of new weapons were added
    • Battle Axe, Blunderbuss, Bolt-Action Rifle, Carbine, Chakram, Fire Axe, Gauntlet, Lance, Machete, Musket, Musket Pistol, Scimitar, Shortsword, Spiked Gauntlet, Throwing Hammer, War Fan
    • 2 new weapon attributes were added as well, Bayonet and Open Hand
  • A number of Feats, Abilities, and weapons were changed slightly as well
    • A complete list of these would be too long to write here, so be sure to check what you have!
  • Some Classes were updated slightly
    • Assassin, Enhancement, Heavyweight, Hive, Independent, Power, Ripple, Spin, Ranger, Revenge, Warrior
  • Passions grant bonus Skill Proficiencies based on your PB now
  • Inspiration was tweaked slightly
    • It now re-rolls the entire Check, rather than just providing Advantage
  • Various smaller fixes and clarifications
    • Familiar Feats now reflect the changed Feat amounts of the last update
    • Those with Berserker active still fail a Death Save when hit
    • Several Feats and Abilities now provide canon examples of their use
Website Update v1.11.7.2
  • Touched up the layout of the Weapons page
    • The table is now scrollable, with the header pinned to the top as you scroll. This should make reading the full list easier.
    • There's now a list of all the weapon attributes below the table, so you can look for a specific attribute rather than trying to find a weapon with it and hovering over it in the list. The tooltips in the list itself still work as well of course.

Update 1.11.7

  • Number of Feats is now equal to your Proficiency Bonus
    • This means you'll get your next Feat sooner and overall will make it easier to keep track of.
  • Berserker was nerfed slightly
    • It now inflicts Disadvantage on all mental Saves, rather than just illusions.
  • Some more small clarifications
    • Energy imbuement grants Proficiency in the imbued item.

Update 1.11.6

  • New Undead Ability - Spider Climb
    • Vampires and EPMs can now walk on sheer surfaces and ignore gravity while on them
    • Basically a more balanced version of the flight they used to have
  • Some small additions and clarifications
    • Updated our Patron list, huge thanks to our 3 supporters! If you want your name changed on the site/doc, please let me know.
    • Immense Resistance has a minimum increase specified now

Update 1.11.5

  • More Non-Supernatural Classes!
    • In addition to those added in v1.11.0, we've added 3 more, the Artisan, Heavyweight, and Ranger!
  • Some existing Classes were tweaked
    • Act 1's stat restritctions have been swapped, they now CAN increase their Speed, but can't increase Durability
    • Assassins, Consuls, Scholars, and Warriors have had some early Abilities shifted slightly.
    • A number of Classes have been given options for some of the newly added Feats and Abilities.
  • A new Feat was added, Berserker
    • This allows you to go into a rage under specific conditions, increasing your stats and giving your attacks and defenses some bonuses.
  • Passions were given a fresh coat of paint
    • This includes some small tweaks as well as a few new Passions, the Chef, Scientist, and Weightlifter!
  • A list of Tools has been added to the Rules page
    • These have always existed for the most part, and a good number of them come from 5e, but we figured it'd be useful for us to specify how they may be used in our system.
  • Some Abilities were tweaked
    • Art of Distraction, Assassin Fighting Style, Damage Minimum, Encouragement Dice, Scholarly Combatant, Reckless Attack, Supernatural Item, Warrior Fighting Style
  • A number of new Weapons were added
    • Bec de Corbin, Club, Greatclub, Hand Crossbow, Heavy Repeating Crossbow, Mace, Quarterstaff, Rapier, Repeating Crossbow, Throwing Dart, Whip
  • Additional clarifications and changes
    • A description of how Light weapons and dual-wielding works has been added to the Rules page.
    • The Completed Stone Mask can only be used on a Vampire or EPM at level 20+.
    • Trained Combatant can be taken by Stand Users now.
  • A couple more small tweaks and fixes

Update 1.11.0 - The "Smaller" Update

  • A clarifying rule was added about the maximum Level
    • Its not necessarily 20, but allowing any character or enemy to go above that and how its handled will be at the DMs discretion
    • More information about this and balancing will be added to the upcoming DM guide!
  • Rock Human's Natural Armor was reworked
  • A Variant Rule for Act 4 was added
  • Several other Abilities were tweaked or clarified:
    • Blow for Blow, Rejuvenation, Spiritual Spin
  • A new Variant Rule option for DMs was added, Epic Feats!
    • These are very strong Feats meant for particularly powerful characters
    • Using these for characters, especially allowing Players to take them, is up to the DM
  • Several other Feats were tweaked or clarified:
    • Calculating, Combat Medic, Consistent Accuracy, Demolitionist, Energy Imbuement, Exemplar, Level-Headed, Stand Specialization (now Anchored Immunity), Stroke of Luck
  • Some new Feats were also added:
    • Defensive Stance (now a Feat rather than an Ability), Duelists Agility, Impeccable Agility, Massive Stand, Shield Proficiency, Uncontrollable Stand
  • Some Class Features were tweaked:
    • Spin and Ripple now get their 'spiritual' Variant Rule Ability sooner
    • Revenge Type gains the option of Rejuvenation at Level 7, rather than Catlike Agility
  • Non-Supernatural Classes were Overhauled, being buffed significantly to better keep up with other Classes to some extent
    • The existing Classes were replaced by new ones, the Assassin, Consul, Scholar, and Warrior
    • Each one gets their own fighting styles and some unique Class Features, and of particular note are the new Supernatural Items they can obtain as a Variant Rule
    • Additionally, multiclassing into both a Non-Supernatural and Supernatural class is now possible
  • A number of Weapons were added/readded, along with some new attributes
    • Items of note include the Atlatl and Dart, more basic 5e-style weapons, and shields
  • All Fandom links have been replaced with links to the proper JoJo Wiki
  • And of course, a number of typo fixes and clarifications
    • Of note, a section was added to the main Stand rules page about their interactions with Skill Checks and Saving Throws
